# README Images

This directory is reserved for public README images captured with sanitized
demo data.

Use these stable filenames when adding images:

- `oriel-hero.png` - Wide hero image shown near the top of the README.
- `oriel-timeline.png` - Timeline workspace with Activity Stream, Time Entries,
  Work Times, date navigation, and zoom/day-week controls.
- `oriel-projects.png` - Projects page or project detail modal with fake
  projects, billing settings, Time History, and manual time/import controls.
- `oriel-ai-insights.png` - AI Insights daily, weekly, and monthly cards with
  sanitized generated or ready-to-generate content.
- `oriel-preferences.png` - Preferences showing Capture & Privacy or AI
  settings with fake values.

Do not commit real activity history, browser URLs, client or project names,
API keys, local machine paths, SQLite files, logs, archives, or screenshots with
private data. Avoid filenames containing `screenshot` because local screenshot
artifacts are ignored by the repository.
